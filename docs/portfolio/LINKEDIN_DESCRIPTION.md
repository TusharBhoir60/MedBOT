Excited to announce the v1.0.0 release of **AarogyaAgent v2**! 🚀🩺

Over the past weeks, I've been building a full-stack, AI-powered clinical triage system designed to solve the bottlenecks in modern healthcare intake. 

Instead of building a simple ChatGPT wrapper, I architected a deterministic **LangGraph Multi-Agent System**. It uses a custom Confidence-Weighted Reasoning (CMAR) model to interact with patients, retrieve real medical guidelines via **RAG**, and generate differential diagnoses. Most importantly, it knows when it's unsure, automatically routing complex cases to a secure Physician Dashboard!

**Tech Stack:**
⚛️ Next.js 16 & Tailwind CSS v4
🐍 FastAPI & PostgreSQL
🧠 LangChain & ChromaDB
🐳 Docker & GitHub Actions (100% Test Coverage!)
