# Project Summary for Portfolio

**Project Title:** AarogyaAgent v2: Autonomous Clinical Triage & Decision Support System

**Short Description:**
A full-stack, AI-powered healthcare platform designed to automate patient intake and triage. Built using Next.js and FastAPI, it orchestrates a LangGraph multi-agent system to safely diagnose patients using RAG against medical guidelines, escalating complex cases to a custom physician dashboard.

**Key Technologies:**
- Next.js 16, React 19, Tailwind CSS v4, TanStack Query
- FastAPI, Python 3.12, SQLAlchemy, PostgreSQL
- LangChain, LangGraph, OpenAI, ChromaDB
- Docker, GitHub Actions, Pytest, Playwright

**My Role:**
*Lead Full-Stack Engineer / AI Architect*
I architected and implemented the entire system from the ground up, establishing the CMAR (Confidence-Weighted Multi-Agent Reasoning) framework, designing the decoupled frontend-backend architecture, and enforcing a 100% test pass-rate CI/CD pipeline.
